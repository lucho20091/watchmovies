# MovieCard Color Palette

This document outlines the specific Tailwind CSS color classes used for each UI element within the `MovieCard` component (`components/MovieCard.jsx`).

-   **Card Background**: `bg-neutral-950`
-   **No Poster Available Background**: `bg-gray-800`
-   **No Poster Available Text**: `text-gray-400`
-   **Gradient Overlay (for title readability)**: `from-neutral-950`, `to-transparent`
-   **Title (always visible)**: `text-white`
-   **Original Language Badge Background**: `bg-black/70`
-   **Original Language Badge Text**: `text-white`
-   **Adult Content Badge Background**: `bg-red-600`
-   **Adult Content Badge Text**: `text-white`
-   **Hover Overlay Background**: `bg-black/80`
-   **Hover Title Text**: `text-white`
-   **Hover Overview Text**: `text-white`
-   **Hover Rating Icon/Text**: `text-yellow-400`
-   **Hover Year Text**: `text-white`
