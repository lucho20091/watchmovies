# MovieCard Color Palette

This document outlines the specific Tailwind CSS color classes used for each UI element within the `MovieCard` component (`components/MovieCard.jsx`).

-   **Card Background**: `bg-neutral-950`
-   **No Poster Available Background**: `bg-neutral-950`
-   **No Poster Available Text**: `text-rich-mahogany-100`
-   **Gradient Overlay (for title readability)**: `from-neutral-950`, `to-transparent`
-   **Title (always visible)**: `text-rich-mahogany-100`
-   **Original Language Badge Background**: `bg-rich-mahogany-950`
-   **Original Language Badge Text**: `text-rich-mahogany-100`
-   **Adult Content Badge Background**: `bg-rich-mahogany-950`
-   **Adult Content Badge Text**: `text-rich-mahogany-100`
-   **Hover Overlay Background**: `bg-black/80`
-   **Hover Title Text**: `text-rich-mahogany-100`
-   **Hover Overview Text**: `text-rich-mahogany-100`
-   **Hover Rating Icon/Text**: `text-yellow-400`
-   **Hover Year Text**: `text-rich-mahogany-100`
