# Movies Page Spacing Rules

This document outlines the specific Tailwind CSS spacing classes used for each UI element within the `MoviesPage` component (`app/movies/page.jsx`).

-   **Main Content Container**: `p-4`, `lg:pt-6`, `md:px-0`
-   **Category Selector Container**: `py-2`, `mb-4`
-   **Category Selector Inner Flex**: `space-x-4`, `px-4`, `md:px-0`
-   **Active Category Underline**: `w-full`, `h-0.5`
-   **Movie Grid Container**: `gap-4`
-   **Pagination Controls Container**: `gap-2`, `mt-16`, `mb-12`
-   **Pagination Buttons**: `px-4`, `py-2`
-   **Suspense Fallback Container**: `p-4`, `md:p-0`, `md:pt-4`
-   **Suspense Fallback Title Placeholder**: `h-10`, `w-64`, `mb-4`, `md:mb-0`
-   **Suspense Fallback Category Selector Placeholder**: `py-2`
-   **Suspense Fallback Category Items Flex**: `gap-4`, `px-4`, `md:px-0`
-   **Suspense Fallback Category Item Placeholder**: `h-8`, `w-28`
-   **Suspense Fallback Loading Text**: `mt-8`
