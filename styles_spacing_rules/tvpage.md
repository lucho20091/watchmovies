# TV Series Episode Page Spacing Rules

This document outlines the specific Tailwind CSS spacing classes used for each UI element within the `TvPageSeasonEpisode` component (`app/tv/[id]/[season]/[episode]/page.jsx`).

-   **Main Content Container**: `pt-4`, `md:pt-10`, `pb-20`, `px-4`
-   **Title**: `mb-4`
-   **Info Text (Year, Rating, Status)**: `gap-4`, `mb-4`
-   **Genre Badges Container**: `mt-4`, `gap-2`, `mb-6`
-   **Overview Text**: `mb-8`
-   **Server Selection Buttons Container**: `gap-4`, `mb-8`
-   **Season Selector**: `mb-4`, `w-36`, `px-2`, `py-1`
-   **Episode Grid Container**: `md:mb-4`, `mt-4`
-   **Episode Grid Inner**: `gap-4`, `p-4`
-   **Episode Buttons**: `px-4`, `py-2`
-   **Video Player Iframe Container**: `mt-8`
