# Homepage Spacing Rules

This document outlines the specific Tailwind CSS spacing classes used for each UI element within the Homepage (`app/page.js`).

-   **Hero Section Container**: `h-[500px]`, `md:h-[700px]`
-   **Hero Section Content Container**: `p-4`, `md:p-8`, `max-w-screen-xl`, `mx-auto`
-   **Hero Section Title**: `mb-4`
-   **Hero Section Overview Text**: `mb-4`
-   **Hero Section Rating/Year Text**: `gap-4`, `mb-6`
-   **Hero Section "Watch Now" Button**: `py-3`, `px-6`
-   **Main Content Container**: `p-4`, `md:px-0`, `mx-auto`
-   **Section Titles (e.g., "TOP 10 MOVIES")**: `mb-8`, `md:mt-8`
-   **Section Title Accent Bar**: `w-2`, `h-8`, `mr-3`
-   **"View More" Button**: `py-2`, `px-4`
-   **Movie Grid Container**: `mt-4`, `md:mt-8`, `gap-4`
-   **Series Grid Container**: `mt-4`, `md:mt-8`, `gap-4`, `md:mb-8`
