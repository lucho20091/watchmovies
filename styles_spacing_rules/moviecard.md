# MovieCard Spacing Rules

This document outlines the specific Tailwind CSS spacing classes used for each UI element within the `MovieCard` component (`components/MovieCard.jsx`).

-   **Main Card Container (`Link`)**: `h-full`
-   **Image Wrapper (`div`)**: `w-full`, `aspect-[2/3]`
-   **No Poster Available Placeholder (`div`)**: `p-2`
-   **Gradient Overlay (`div`)**: `h-1/2`
-   **Always Visible Title Container (`div`)**: `p-3`
-   **Original Language Badge (`div`)**: `top-2`, `right-2`, `px-2`, `py-1`
-   **Adult Content Badge (`div`)**: `top-2`, `left-2`, `px-2`, `py-1`
-   **Hover Overlay (`div`)**: `p-4`
-   **Hover Title (`h3`)**: `mb-2`
-   **Hover Overview Text (`p`)**: `mb-2`
-   **Hover Rating (`div`)**: `gap-1`, `mb-2`
