# Movie Details Page Spacing Rules

This document outlines the specific Tailwind CSS spacing classes used for each UI element within the `MoviePage` component (`app/movie/[id]/page.jsx`).

-   **Main Content Container**: `p-4`, `md:py-10`, `pb-20`
-   **Title**: `mb-4`
-   **Info Text (Year, Rating, Runtime)**: `gap-4`, `mb-4`
-   **Genre Badges Container**: `mt-4`, `gap-2`, `mb-6`
-   **Overview Text**: `mb-8`
-   **Server Selection Buttons Container**: `gap-4`, `mb-8`
-   **Server Selection Buttons**: `px-6`, `py-3`
-   **Video Player Iframe Container**: `mt-8`
